## 🛠 Deploy (GitHub Pages)

1. Push this repo to GitHub.
2. **Settings → Pages → Source:** deploy from `main` / root.
3. (Optional) Add a custom domain `arasghorbani.com` in **Pages → Custom domain**; create a `CNAME` file with that domain.
4. Live at `https://arasghorbani9090.github.io/arasghorbani9090-web/` (or your custom domain).

> For a root URL like `https://arasghorbani9090.github.io`, rename the repo to `arasghorbani9090.github.io`.
